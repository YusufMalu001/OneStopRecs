Available Datasets

GET
/api/datasets
Get information about available datasets and their models
curl http://localhost:5000/api/datasets

Recommendations

GET
/api/recommend/{dataset}/{model}
Get personalized recommendations for a user
curl "http://localhost:5000/api/recommend/movies/svd?user_id=1&n=10"
GET
/api/recommend-by-categories/{dataset}/{model}
Get recommendations filtered by categories/genres
curl "http://localhost:5000/api/recommend-by-categories/movies/svd?user_id=1&categories=Action&categories=Drama&n=5"

Rating Prediction

GET
/api/predict/{dataset}/{model}
Predict rating for a user-item pair
curl "http://localhost:5000/api/predict/movies/svd?user_id=1&item_id=100"

Search & Browse

GET
/api/search/{dataset}
Search for items in a dataset
curl "http://localhost:5000/api/search/movies?q=star&limit=10"
GET
/api/categories/{dataset}
Get available categories/genres for a dataset
curl http://localhost:5000/api/categories/movies

Model Information

GET
/api/model-info/{dataset}/{model}
Get detailed information about a specific model
curl http://localhost:5000/api/model-info/movies/svd

Frontend Integration

This API is designed to work with the React frontend. The frontend can:

  • Select categories (Movies, Books, Products)
  • Choose specific genres/tags
  • Get personalized recommendations
  • Search for specific items
  • View item details and predicted ratings

Frontend URL: http://localhost:3000
